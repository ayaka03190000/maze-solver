# maze-solver
ターミナルで動く迷路作成ツールです。C++で書かれています。

## デモ

## 機能
迷路の大きさとスタート位置を指定すると自動で迷路が作られます。
ゴール位置は、スタート位置から最も道のりが長い場所に設定されます。
作成した迷路が表示された後Enterを押すと、迷路に`o`で解答が表示されます。

## 必要条件
- C++コンパイラ(g++)
- makeコマンド

## 使用方法
### １.ビルド
以下を実行して`maze-solver`という実行ファイルを作ります。
```bash
make
```

### ２.実行
```bash
./maze-solver
```

### ３.迷路の大きさを指定する
迷路の横の長さを指定する。
```
迷路の横の長さ=
```
迷路の縦の長さを指定する。
```
