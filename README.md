# maze-solver
ターミナルで動く迷路作成・探索ツールです。C++で実装しています。

## 機能
- 迷路の大きさとスタート位置を指定すると、自動で迷路が生成されます。
- ゴール位置は、スタート位置から最も道のりが長い場所に設定されます。
- 生成された迷路の表示後Enterを押すと、迷路に`o`で解答が表示されます。

## 特徴
- DFSを用いて迷路生成を行い、ランダムに探索順を変えることで多様な迷路を作成しました。
- BFSを用いて経路探索を行い、スタートからゴールまでの最短経路を求め、必ず最短ルートが表示されるようにしました。
- 2次元配列を用いて迷路を管理し、visited配列で重複探索を防止しました。

## 必要条件
- C++コンパイラ(g++)
- makeコマンド

## 使用方法
### １.ビルド
以下を実行して`maze-solver`という実行ファイルを作ります。
```bash
make
```

### ２.実行
```bash
./maze-solver
```

### ３.迷路の大きさを指定する
迷路の横の長さを指定します。
```
迷路の横の長さ=
```
迷路の縦の長さを指定します。
```
迷路の縦の長さ=
```

### ４.スタート位置を指定する
スタート位置を左から何番目にするか指定します。
```
スタート位置x（左から何番目）=
```
スタート位置を上から何番目にするか指定します。
```
スタート位置y（上から何番目）=
```

### ５.解答の表示
スタート位置を指定した後迷路が生成され、その後Enterを押すと解答を表示できます。
