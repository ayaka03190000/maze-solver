# maze-solver
ターミナルで動く迷路作成ツールです。C++で書かれています。

## デモ

## 機能
迷路の大きさとスタート位置を指定すると自動で迷路が作られます。
ゴール位置は、スタート位置から最も道のりが長い場所に設定されます。
作成した迷路が表示された後Enterを押すと、迷路に`o`で解答が表示されます。

## 必要条件
- C++コンパイラ(g++)
- makeコマンド

## 使用方法
### １.ビルド
以下を実行して`maze-solver`という実行ファイルを作ります。
```bash
make
```

### ２.実行
```bash
./maze-solver
```

### ３.迷路の大きさを指定する
迷路の横の長さを指定する。
```
迷路の横の長さ=
```
迷路の縦の長さを指定する。
```
迷路の縦の長さ=
```

### ４.スタート位置を指定する
スタート位置を左から何番目にするか指定する。
```
スタート位置x（左から何番目）=
```
スタート位置を上から何番目にするか指定する。
```
スタート位置y（上から何番目）=
```

### ５.解答の表示
スタート位置を指定した後迷路が生成され、その後Enterを押すと解答を表示できる。
